# 🎨 UI Button Design System

This project showcases a collection of modern and interactive button components designed using Figma as part of a UI/UX internship assignment.

## 🚀 Overview

The goal of this task is to demonstrate strong UI fundamentals, visual consistency, and interaction design through multiple button styles, hover effects, and modern UI trends.

---

## 🔘 Button Variants

### 🔹 Button 1 – Primary Button (Hover Variant)

* Default state: Solid blue button
* Hover state: Lighter blue variation
* Focus: Basic interaction and color transition

---

### 🔹 Button 2 – Layered Depth Button

* Default state: Flat light background
* Hover state: Layered shadow effect appears beneath
* Focus: Depth, elevation, and visual hierarchy

---

### 🔹 Button 3 – Animated Stroke Button (Snake Effect)

* Default state: Minimal light background
* Hover state: Animated stroke moves around the border like a snake
* Focus: Micro-interaction and motion design
* Highlights:

  * Continuous border animation
  * Enhances user engagement
  * Modern interactive feedback

---

### 🔹 Button 4 – Animated Gradient Button

* Default state: Dark button with neon glow
* Hover/active: Gradient glow effect intensifies
* Focus: Modern UI trends and dynamic visual appeal

---

### 🔹 Button 5 – Glassmorphism Button

* Transparent glass-like surface
* Subtle blur with soft border and shadow
* Focus: Trend-based UI styling (Glass UI)

---

## ✨ Features

* Interactive hover states for all buttons
* Micro-interactions (stroke animation, glow effects)
* Modern UI styles (Gradient, Glassmorphism)
* Consistent spacing, typography, and alignment
* Component-based design approach
